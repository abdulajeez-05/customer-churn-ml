# customer-churn-ml
Machine learning project to predict customer churn using Logistic Regression, Decision Tree, and Random Forest with focus on recall and business insights


# Customer Churn Prediction

## Overview
Built ML models to predict customer churn.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest

## Key Learnings
- Importance of recall in churn prediction
- Threshold tuning
- Feature importance analysis
- Model comparison

## Result
Logistic Regression performed best with high recall.

## Business Insight
New customers with high monthly charges are more likely to churn.
